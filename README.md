## Setup instructions

Install the following R packages:

```
perishky/meffonym
data.table
R.utils
ggplot2
ggrepel
readxl
here
```

Next create a `config.env` file based on the `config-template.env` template.

## To run

You can run the entire pipeline using 

```
bash run-all.sh
```
